# MiMo Cost Optimizer

Smart conversation management, on-demand prompt loading, and usage monitoring for MiMo API cost optimization.

## Features

- **Smart Conversation Manager** — topic detection, token compression, round efficiency checking
- **Prompt Loader** — on-demand domain-specific prompt loading
- **Usage Monitor** — real token data from Hermes database

## Quick Start

```bash
python3 test_optimization.py  # Run all tests
python3 mimo_usage_real.py 7  # View usage for last 7 days
```
